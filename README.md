# EchoLocation

A 2.5D exploration game built with Godot.

## Getting Started

1. Open `project.godot` in Godot 4.3+
2. Press F5 to run
3. Use arrow keys/WASD to move, Space to jump

## Scenes

- **Main.tscn** - Main game area (production)
- **TestArea.tscn** - Development/test area for experimentation

### Switching Between Scenes

To test different scenes:
1. In Godot, go to **Project > Project Settings > Application > Run > Main Scene**
2. Change the main scene to the one you want to test
3. Or right-click a scene file and select "Set as Main Scene"

Or simply:
- Right-click `scenes/TestArea.tscn` → "Set as Main Scene" for testing
- Right-click `scenes/Main.tscn` → "Set as Main Scene" for production
